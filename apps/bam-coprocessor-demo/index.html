BAM coprocessor proof verification demo

EIP-4844 blobs expire from Ethereum's consensus layer in roughly 18 days. After that, verifying any message from the blob requires trusting whoever ran the pipeline.

This demo shows a ZK proof that runs the full BAM reader pipeline inside the SP1 zkVM — KZG verification, ECDSA signature checking, and message decoding. The proof is 1.8 KB and can be verified locally in your browser. No trust in the server required.
