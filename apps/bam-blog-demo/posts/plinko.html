Plinko PIR tutorial

2025-11-25

One underrated form of privacy, that is not well satisfied by ZK-SNARKs, traditional versions of FHE or other commonly talked about methods, is private reads. Users want to learn things from a large database, without revealing what they're reading.

There is a category of privacy protocols that are directly designed to solve this problem, called private information retrieval (PIR). A server has a database D. A client has an index i. The client sends the server a query, the server replies back with an answer that allows the client to reconstruct D[i], all without the server learning anything about i.
